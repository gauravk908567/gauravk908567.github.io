Furry Escape

Furry Escape is a 2D platformer game designed to create a challenging yet enjoyable hyper-casual gaming experience. The game is available on both the Android platform and Itch.io. This project marks my first foray into WebGL game development, providing valuable insights into Unity 2D and the deployment process. The game features various platforms with unique behaviors and utilizes Player Prefs to save high scores, enhancing player engagement.

Objectives of the Project:

  • Game State Management: Utilized Player Prefs to save game state and high scores, allowing players to track their progress.
  • Scrolling Mechanics: Implemented infinite scrolling using parallax scrolling techniques to create a dynamic and immersive gameplay experience.
  • Visual Asset Creation: Designed and created visually appealing game assets and environments using Adobe Photoshop and Illustrator, ensuring a cohesive aesthetic.
  • Challenging Game Mechanics: Developed engaging and challenging game mechanics, including character movement and platform interactions, leveraging Unity's 2D Physics Engine and custom scripts.
  • Unique Platform Interactions: Implemented unique platform interactions, such as: Breaking platforms after player collision, adding unpredictability to gameplay. Pushing players in indicated directions, requiring strategic thinking. Following player movement to create tension and increase difficulty.
  • Game Publishing: Successfully published the game on the Play Store and Itch.io by creating both Android and WebGL builds.
  • Version Control: Employed Git for version control to track project progress and maintain access to previous iterations.

  • Breaking Platforms: Platforms break after player collision, introducing a level of unpredictability and challenging players to adapt their strategies.
  • Directional Pushing: Platforms push players in indicated directions, requiring players to remain aware of their surroundings and adjust their tactics accordingly.
  • Movement Following: Platforms that follow player movement can trick players into falling into spikes, increasing tension and difficulty.
